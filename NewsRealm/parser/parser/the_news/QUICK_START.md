# Быстрый старт

## Создание базы данных и пользователя

### 1. Создайте пользователя и базу данных

```bash
sudo -u postgres psql -f create_user_and_database.sql
```

**ВАЖНО:** Перед выполнением измените пароль в файле `create_user_and_database.sql` на более безопасный!

### 2. Создайте таблицы и индексы

```bash
psql -U news_database_admin -d the_news -f create_database.sql
```

### 3. Обновите настройки в settings.py

Откройте файл `the_news/settings.py` и измените пароль:

```python
PG_PASSWORD = "ваш_безопасный_пароль"
```

### 4. Запустите парсер

```bash
cd the_news
scrapy crawl rbc_news
```

## Что такое индексы?

**Индексы** - это структуры данных, которые ускоряют поиск в базе данных.

**Аналогия:** Представьте библиотеку:
- **Без индекса** = искать книгу, просматривая все полки подряд
- **С индексом** = использовать каталог, чтобы сразу найти нужную полку

**В нашей базе:**
- Индексы ускоряют поиск по категориям, датам, языку и другим полям
- Запросы типа `WHERE category = 'politics'` выполняются намного быстрее
- Небольшой недостаток: индексы занимают место и немного замедляют добавление новых записей

Подробнее см. `DATABASE_SETUP.md`
